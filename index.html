🔐 Discord Token取得ブックマークレット

⚠️ 重要な注意事項（2025年版）

このツールは自分のアカウントのトークンを取得するためのものです。

  • 取得したトークンは絶対に他人に教えないでください
  • トークンが漏洩すると、アカウントが乗っ取られる可能性があります
  • 開発・テスト目的でのみ使用してください
  • Discordの利用規約を遵守してください
  • 2024年以降、Discordのセキュリティ対策が強化されています
  • 一部のブラウザでブックマークレットが制限される場合があります

📱 モバイル対応版

モバイルブラウザ用の手順：

📌 Chrome Mobile（Android）

  1. 任意のページをブックマークに追加
  2. ブックマークを編集
  3. URLの部分を以下のコードに置き換える：
javascript:(function(){try{const token=localStorage.getItem('token')||sessionStorage.getItem('token');if(token){const cleanToken=token.replace(/["]/g,'');const result=prompt('📋 以下のトークンをコピーしてください:',cleanToken);if(result){console.log('Token copied to clipboard area');}}else{alert('❌ トークンが見つかりませんでした。');}}catch(e){alert('⚠️ エラー: '+e.message);}})();

📌 Safari Mobile（iOS）

🚨 2025年版の追加対策について

⚠️ Discordのセキュリティ強化により、以下の制限があります：
• 一部のブラウザでブックマークレットがブロックされる場合があります
• Content Security Policy（CSP）により実行が制限される場合があります
• 動作しない場合は、開発者ツールでの手動取得をお試しください
• 拡張機能の使用も検討してください（Chrome Web Store等）

🛠️ 手動での取得方法

1️⃣ Discord（Web版）でF12を押して開発者ツールを開く
2️⃣ Consoleタブをクリック
3️⃣ 以下のコードを貼り付けて実行：
// トークンを取得してコピー const token = localStorage.getItem('token') || sessionStorage.getItem('token'); if (token) { const cleanToken = token.replace(/["]/g, ''); console.log('Your Discord Token:', cleanToken); navigator.clipboard.writeText(cleanToken); console.log('✅ トークンをクリップボードにコピーしました！'); } else { console.log('❌ トークンが見つかりませんでした'); }

🔒 セキュリティについて

取得したトークンは以下の点にご注意ください：

  • トークンは「パスワード」と同じくらい重要です
  • GitHub等の公開リポジトリに絶対にアップロードしないでください
  • 不審な活動を発見した場合は、すぐにパスワードを変更してください
  • 定期的にアカウントのセキュリティを確認してください
